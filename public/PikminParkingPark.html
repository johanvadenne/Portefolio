Pikmin Parking Park

Description

Pikmin Parking Map est un projet passionnant qui a pour objectif de simplifier la recherche de places de stationnement pour les personnes handicapées. Nous avons donné une touche amusante à cette tâche en utilisant les adorables Pikmin comme marqueurs sur notre carte.

Demande

Le projet vise à nous faire découvrir Leaflet, une bibliothèque JavaScript largement utilisée notamment pour la cartographie.

Visualisation

Chaque Pikmin représente une place réservée aux personnes handicapées.


Lorsque vous cliquez sur un Pikmin, celui-ci vous indique le nom de la rue correspondante.
